# PASS Guide
1. This is where you take Pairs of Tokens and Copy the Rule Basically

RULE:A# #A → NOTHING

RULE:A# #B → #B A#
NEXT INDEX:+2

NEXT INDEX:+2

NEXT INDEX:+2
RULE:EVERY OTHER RULE -> FIRST TOKEN
NEXT INDEX:+1


# GATHER NEW TOKENS Guide
1. This is Where you Get the New Tokens. Basically Everything After →.
2. Be SURE to Ignore the NOTHING Meaning Literally NOTHING
3. Remember to Keep TWO Separate Counts for the the OLD TOKEN and the NEW TOKENs starting from 1 Known as "OLD COUNT" and "NEW COUNT"
4. AGAIN ALWAYS Start the Count from 1 for OLD TOKENs and NEW TOKENs



# EXACT SAME STRING PREV Guide
1. This is Where Check if the String is the Same so you can just go to "# Tokens Without Numbers" then Submit your Solution on last line
2. Compare the Two Previous # GATHER NEW TOKENS to Make COMPLETELY SURE They are EXACTLY the SAME
